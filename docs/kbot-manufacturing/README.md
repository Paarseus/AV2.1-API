# KBot Manufacturing Documentation

Reference documentation for manufacturing K-Scale KBot humanoid robot parts using CNC machining services.

---

## Documentation Index

| Document | Description |
|----------|-------------|
| [CNC Manufacturing Guide](kbot-cnc-manufacturing-guide.md) | Comprehensive guide covering all aspects of CNC manufacturing for KBot |
| [RobStride Actuator Specs](robstride-actuator-specs.md) | Technical specifications for all RobStride actuators used in KBot |
| [JLCCNC Ordering Checklist](jlccnc-ordering-checklist.md) | Step-by-step checklist for ordering from JLCCNC |
| [Arm BOM](kbot-arm-bom.md) | Bill of materials template for one KBot arm |
| [Cost Estimation](cnc-cost-estimation.md) | Worksheet for estimating CNC manufacturing costs |

---

## Quick Start

1. **Review specifications**: Start with `robstride-actuator-specs.md` to understand mounting requirements
2. **Plan your order**: Use `kbot-arm-bom.md` as a template for parts list
3. **Estimate costs**: Use `cnc-cost-estimation.md` to budget
4. **Place order**: Follow `jlccnc-ordering-checklist.md` step by step
